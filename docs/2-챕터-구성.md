# 챕터 구성

- Version: 1.0
- Date: 2026-04-17
- Project id: `zunyang-party-panic`
- Purpose: 조력자 시점 기준으로 각 챕터에서 플레이어가 실제로 무엇을 하게 되는지 다시 정리한다.

## 1. 공통 규칙

- 플레이어는 스트리머 본인이 아니라 `백스테이지 조력자`를 직접 조작한다
- 각 챕터는 `타일 이동 -> 조사 -> 짧은 대사/연출 -> 미니게임 또는 정리 파트 -> 허브 변화 확인` 흐름을 기본으로 한다
- 무엇을 하든 `오늘 밤 무대를 여는 준비`라는 축은 계속 살아 있어야 한다
- 조력자는 사람의 속마음을 대신 말하지 않고, 공간과 물건에서 읽히는 분위기를 정리한다
- 이동은 `4방향`만 사용한다
- 조사 가능한 오브젝트는 맵 위 한 칸 이벤트로 둔다
- 챕터를 마칠 때마다 허브의 일부가 실제로 달라져야 한다
- 스트리머는 드물게 등장하는 NPC나 오프스크린 음성으로만 사용하고, 모든 장면의 중심을 가져가지 않게 한다

## 2. 챕터 0: 프롤로그 - 호출등이 켜진 준비동

### 목적

- 플레이어가 `오늘 밤을 열 준비를 맡은 조력자`라는 사실을 바로 전달한다
- 준비동이 아직 미완성 상태라는 걸 보여 준다
- 첫 목표를 `큐 부스`로 유도한다

### 콘텐츠

1. 호출등 점등 연출
2. 기본 이동
3. 준비동 주요 오브젝트 첫 조사
4. 잠긴 도어 확인
5. 첫 체크리스트 입수
6. 큐 부스로 유도

### 조사 포인트

- 큐 부스
- 중앙 테이블
- 포토 베이
- 메시지 월
- 소품 아카이브 도어
- 기록 복도 도어

### 연출 기준

- 오프닝 정보는 컷신보다 `짧은 호출 + 조사 반응`으로 푼다
- 세계관 설명을 길게 하지 않고, 미완성 상태가 플레이에서 보이게 한다
- 첫 목표는 화려한 연출보다 `실무적인 급함`으로 제시하는 편이 맞다

### 끝날 때의 느낌

`좋아. 먼저 첫 신호부터 안정시키자.`

## 3. 챕터 1: 첫 신호를 맞추다

### 목적

- 스트리머가 자리에 앉자마자 흔들리지 않도록 첫 화면과 첫 호흡을 정리한다
- 조력자 플레이의 첫 만족감을 `세팅 완료`에서 느끼게 만든다

### 콘텐츠

1. 큐 부스 주변 조사
2. 마이크, 조명, 모니터 체크
3. 오프닝 큐시트 확인
4. `첫 신호 맞추기` 미니게임
5. 큐 램프 점등

### 핵심 오브젝트

- 마이크
- 모니터
- 큐시트
- 데스크 램프
- 타이머
- 예비 배터리

### 미니게임 방향

- `방송을 직접 진행하는 체험`이 아니라 `오프닝 세팅을 맞추는 체험`이어야 한다
- 볼륨, 빛, 화면 타이밍, 오프닝 큐를 조율하는 감각이 핵심이다
- 결과는 점수보다 `첫 인사가 흔들리지 않는 상태`로 느껴져야 한다

### 끝날 때의 느낌

`이제 자리에 앉기만 하면 첫 호흡은 바로 이어진다.`

## 4. 챕터 2: 빈 자리를 채울 것들

### 목적

- 무대에 아직 비어 있는 자리를 채울 재료를 회수한다
- 조력자의 선별 감각과 실무 감각을 보여 준다

### 콘텐츠

1. 소품 아카이브 잠금 해제
2. 아카이브 진입
3. 필요한 소품 조사
4. 회수 우선순위 판단
5. 간단한 정리 퍼즐
6. 허브로 복귀

### 핵심 오브젝트

- 리본 상자
- 조명 젤
- 케이크 토퍼
- 예비 천
- 소형 소품 바구니
- 기록용 장식 소품

### 연출 기준

- 전부 다 챙기는 구조보다 `오늘 필요한 것만 고르는 구조`가 맞다
- 조력자가 공간을 정리하는 동안 허브 쪽 변화를 기대하게 만들어야 한다
- 이 챕터가 끝나면 허브가 `조금 더 오늘답게` 보이기 시작해야 한다

### 끝날 때의 느낌

`비어 있는 자리를 채울 건 손에 들어왔다.`

## 5. 챕터 3: 오늘의 중심을 세우다

### 목적

- 화면 가운데에 놓일 생일의 중심 장면을 완성한다
- 준비동 전체가 `오늘 밤 무대`로 읽히게 만드는 전환점을 만든다

### 콘텐츠

1. 중앙 테이블 조사
2. 가져온 소품 적용
3. 배치 순서 확인
4. 중심 균형 미니게임
5. 완성 컷 확인

### 핵심 오브젝트

- 케이크 베이스
- 토퍼
- 촛불
- 리본
- 테이블 천
- 주변 장식 소품

### 미니게임 방향

- `사고를 막는 게임`보다 `완성도를 올리는 게임`에 가깝게 만든다
- 테이블 중앙이 정돈될수록 허브 전체의 인상이 달라지는 감각이 중요하다
- 미세한 배치나 균형을 잡는 손맛이 있으면 좋다

### 끝날 때의 느낌

`이제 화면 가운데가 비어 보이지 않는다.`

## 6. 챕터 4: 남을 장면을 고정하다

### 목적

- 오늘 밤이 끝난 뒤에도 남게 될 한 장면을 준비한다
- 포토 베이를 `장식`이 아니라 `기록 장치`로 느끼게 만든다

### 콘텐츠

1. 포토 베이 조사
2. 배경과 소품 배치
3. 라이트 조절
4. 프레임 정렬 미니게임
5. 결과 컷 확인

### 핵심 오브젝트

- 카메라
- 삼각대
- 의자 또는 스툴
- 배경 천
- 키 라이트
- 바닥 장식

### 미니게임 방향

- 고득점 경쟁보다 `지금 남기고 싶은 장면을 고정하는 감각`이 우선이다
- 빛, 프레임, 시선선이 맞아 들어가면 만족감이 나야 한다
- 결과 컷은 허브 완성도의 일부처럼 느껴져야 한다

### 끝날 때의 느낌

`오늘을 남길 장면 하나는 제대로 잡혔다.`

## 7. 챕터 5: 지난 밤들의 인수인계

### 목적

- 지금의 준비가 처음이 아니라 이어져 온 무대 위에 있다는 걸 보여 준다
- 실무 파트에서 감정 파트로 자연스럽게 넘어간다

### 콘텐츠

1. 기록 복도 개방
2. 복도 탐색
3. 인수인계 단서 3개 회수
4. 오래된 준비 흔적 조사
5. 메시지 월 챕터로 연결

### 핵심 오브젝트

- 낡은 큐시트
- 예전 사진 프레임
- 메모 보드
- 프로젝터
- 이전 출입 패스
- 포장 흔적이 남은 상자

### 기록 사용 방향

- `첫 생일`, `쌓여 온 밤`, `가장 가까운 지난 밤` 같은 감각층으로 나눈다
- 정확한 연도나 사건 정리보다 `무대가 누적돼 왔다`는 실감이 중요하다
- 과거를 길게 설명하지 않고 짧은 조각으로 읽히게 한다

### 끝날 때의 느낌

`이 밤도, 그냥 갑자기 생긴 무대는 아니었다.`

## 8. 챕터 6: 전해질 말을 고르다

### 목적

- 오늘 무대 곁에 어떤 문장을 남길지 정한다
- 응원과 기록을 `읽는 행위`에서 `정리해 두는 행위`로 바꾼다

### 콘텐츠

1. 메시지 월 앞 망설임
2. 첫 메시지 열람
3. 메시지 3~5개 추가 확인
4. 남길 문장 1~2개 선택
5. 마무리 정리
6. 마지막 도어 개방

### 메시지 처리 원칙

- 실제 자료 기반이더라도 길게 복붙하지 않는다
- 닉네임, 개인 정보, 사적인 내용은 전부 정리한다
- 인게임에서는 `오래 붙어 있던 문장`, `남겨진 메시지`, `전해 둘 말`처럼 보이게 한다
- 여러 페이지로 넘겨 읽는 구조가 맞지만, 핵심은 `선택과 정리`다

### 끝날 때의 느낌

`이제 무대 곁에 둘 말도 정리됐다.`

## 9. 챕터 7: 불이 켜지기 직전

### 목적

- 조력자 시점의 서사를 가장 선명하게 완성한다
- 플레이어가 무대의 한가운데가 아니라 `무대를 열어 주는 자리`에 서게 만든다

### 콘텐츠

1. 마지막 도어 개방
2. 완성된 준비동 최종 점검
3. 메인 스테이지 진입
4. 카운트다운 램프 점등
5. 스트리머 짧은 입장
6. 큐 사인
7. 엔딩 카드

### 엔딩 분기 기준

- 세팅 완성도
- 핵심 챕터 수행 밀도
- 기록 회수 깊이
- 메시지 정리 정도

성공/실패보다 `얼마나 좋은 밤으로 출발시키는가`의 차이로 분기하는 편이 맞다.

### 마지막 장면의 기준

- 스트리머가 무대 중앙에 선다
- 조력자는 화면의 가장자리에 있거나, 이미 한 발 물러난 상태다
- 플레이어가 마지막으로 하는 일은 `말하기`보다 `신호를 보내기`에 가깝다

### 끝날 때의 느낌

`좋아. 이제 불을 켜도 된다.`
