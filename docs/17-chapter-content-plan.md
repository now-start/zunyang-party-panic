- Purpose: 100분 분량 기준으로 각 챕터별 실제 콘텐츠 구성 정리

## 1. 기본 원칙

이 문서는 `각 챕터에서 플레이어가 실제로 무엇을 하게 되는가`를 정리한다.

공통 기준:

- 플레이어는 치즈냥을 직접 조작한다
- 인게임에서는 연도를 직접 말하지 않는다
- 과거 자료는 `예전 생일의 기억`, `남아 있는 기록`, `기억 조각`으로 처리한다
- 챕터는 `탐색 -> 조사 -> 짧은 이벤트 -> 미니게임 또는 정리 파트 -> 감정 독백` 흐름을 기본으로 한다
- 실제 인물의 속마음을 작가가 단정하는 문장은 최대한 피한다
- 텍스트는 `관찰 가능한 상황`, `넓게 공감 가능한 반응`, `열린 해석` 쪽으로 쓴다
- 무엇을 하든 `오늘 생일 방송을 연다`는 축은 계속 살아 있어야 한다

## 3. 챕터 0: 프롤로그 - 불이 덜 켜진 준비방

### 챕터 목적

- 오늘이 생일이라는 사실을 알린다
- 준비방이 아직 덜 완성된 상태임을 보여 준다
- 치즈냥이 시작을 망설이고 있음을 드러낸다
- 첫 목표를 `방송 책상`으로 유도한다
- 이 게임의 현재 목적이 `생일 방송 시작`이라는 걸 알려 준다

### 플레이 콘텐츠

1. 오프닝 독백
2. 기본 이동 튜토리얼
3. 준비방 4개 주요 오브젝트 첫 조사
4. 잠긴 문 2개 확인
5. 첫 목표 문구 출력

### 조사 포인트

- 방송 책상
- 케이크 테이블
- 포토존
- 팬레터 우편함
- 장식 창고 문
- 백스테이지 문

### 조사 텍스트 방향

- 설명문보다 치즈냥 반응이 먼저
- `오늘이 왔다`는 실감은 주되, 구체적인 내면 단정은 피한다
- `준비가 아직 덜 됐다`, `여기부터 정리하면 되겠다` 같은 열린 표현이 안전하다

### 챕터 종료 조건

- 방송 책상을 조사하고 상호작용 시작

### 챕터 종료 감정

- 아직 망설이지만, 최소한 첫걸음은 떼었다

## 4. 챕터 1: 방송 책상의 불을 켜다

### 챕터 목적

- 치즈냥이 가장 익숙한 자리로 돌아오게 만든다
- 플레이어에게 첫 완성 경험을 준다
- 오늘 하루의 기본 무드를 정한다
- `스트리머로서의 치즈냥`을 가장 먼저 살린다

### 플레이 콘텐츠

1. 책상 주변 조사
2. 어질러진 물건 정리
3. 오늘의 분위기 선택 이벤트
4. 방송 책상 정리 미니게임
5. 책상 조명 점등 컷

### 세부 콘텐츠

#### A. 책상 조사

- 마이크
- 컵 또는 텀블러
- 키보드
- 메모지
- 작은 장식 인형
- 비어 있는 모니터 화면
- 오프닝 멘트 메모
- 화면 전환용 버튼 패널

#### B. 선택 이벤트

`오늘 어떤 느낌으로 시작하고 싶지?`

선택 예시:

- 차분하고 따뜻하게
- 귀엽고 말랑하게
- 반짝이고 들뜬 느낌으로

이 선택은 이후 책상 주변 색감, 짧은 독백, 일부 UI 문구에 반영된다.

#### C. 미니게임 방향

- 기존 방송 책상 화면을 `채팅/투표`가 아닌 `책상 정리/무드 세팅`형으로 교체
- 시간 압박보다는 순서 정리, 배치, 선택의 감각이 중심
- 정리 결과가 `오늘 방송의 첫 화면`처럼 느껴져야 한다

### 기억 조각 연결

- 첫 기억 조각 1개 개방
- 내용은 `처음 크게 축하받았을 때의 떨림`
- 연도 표기는 없고, 오래된 화면빛/짧은 문장/장면 조각으로 처리

### 챕터 종료 조건

- 책상 미니게임 완료

### 챕터 종료 감정

- `좋아, 적어도 여기서는 나답게 시작할 수 있겠다`

## 5. 챕터 2: 장식 창고를 열다

### 챕터 목적

- 공간을 확장한다
- 장편 구조로 들어간다는 느낌을 준다
- 치즈냥의 취향과 감각을 보여 준다

### 플레이 콘텐츠

1. 창고 문을 여는 조건 확인
2. 열쇠 또는 스위치 역할을 하는 준비방 단서 찾기
3. 장식 창고 진입
4. 필요한 소품 선별
5. 장식 정리 퍼즐
6. 아이템 확보 후 허브 변화

### 핵심 오브젝트

- 리본 상자
- 전구 꾸러미
- 초 상자
- 인형 바구니
- 오래된 생일 배너
- 포장되지 않은 선물 상자

### 퍼즐 방향

- `오늘 쓸 것`과 `추억으로 남길 것`을 나누는 정리 퍼즐
- 또는 색/형태에 맞춰 장식을 걸어 두는 간단 배치 퍼즐

### 감정 역할

- 치즈냥의 감각이 드러나는 챕터
- 준비를 `일`처럼 하는 게 아니라 `마음에 드는 걸 고르는 행위`처럼 느껴져야 함
- 취향은 보여 주되, 실제 인물의 성격을 확정하는 식으로 몰아가지 않는다

### 기억 조각 연결

- 쌓여 온 기억 조각 1개 개방
- 내용은 `해가 바뀌어도 이어져 온 애정`

### 챕터 종료 보상

- 케이크 장식 재료
- 포토존 소품
- 무대용 조명 일부

### 챕터 종료 감정

- `아, 맞아. 나는 이런 걸 고르는 걸 좋아했지`

## 6. 챕터 3: 케이크가 가운데에 놓이는 순간

### 챕터 목적

- 치즈냥이 오늘이 특별한 날임을 피하지 않게 만든다
- 생일의 중심 오브젝트를 완성한다
- `방송 안에서 보여질 오늘의 중심 장면`을 완성한다

### 플레이 콘텐츠

1. 케이크 테이블 조사
2. 창고에서 가져온 재료 적용
3. 케이크 밸런스 미니게임
4. 케이크 완성 컷
5. 짧은 정적과 독백

### 조사 포인트

- 비어 있는 초 자리
- 기울어진 장식
- 아직 묶이지 않은 리본
- 잘못 놓인 접시
- 덜 켜진 촛불 조명

### 미니게임 방향

- 기존 `CakeTableScreen` 유지 가능
- 다만 의미를 `사고 수습`보다 `완성 직전의 균형 잡기`로 바꿈

### 기억 조각 연결

- 첫 기억 또는 가장 가까운 기억 중 하나가 짧게 스친다
- 케이크를 바라보는 순간 과거의 축하 장면 일부가 겹쳐 보이는 연출

### 챕터 종료 조건

- 케이크 완성

### 챕터 종료 감정

- `부끄럽긴 한데... 싫진 않다. 오늘은 진짜 생일이구나`

## 7. 챕터 4: 포토존에 오늘을 남기다

### 챕터 목적

- 오늘을 기록하고 싶어지는 마음을 만든다
- 분위기를 한 단계 밝힌다
- 방송 화면에 남길 장면을 직접 고르게 만든다

### 플레이 콘텐츠

1. 포토존 조사
2. 소품 배치
3. 조명 조정
4. 포토 미니게임
5. 결과 컷 확인

### 조사 포인트

- 카메라 삼각대
- 말랑한 소품 인형
- 의자 또는 스탠드
- 배경 천
- 반쯤 켜진 조명
- 바닥에 떨어진 장식

### 이벤트 포인트

- 소품 2~3개 중 어떤 걸 전면에 둘지 선택
- 표정이나 포즈 관련 짧은 독백
- 방송 썸네일/기념 장면처럼 보일 컷을 의식하게 만드는 문구

### 미니게임 방향

- 기존 `PhotoTimeScreen` 유지 가능
- 의미를 `좋은 점수`보다 `마음에 드는 컷 한 장 남기기`로 조정

### 기억 조각 연결

- 가장 가까운 기억 조각 1개 개방
- 영상 프레임이나 짧은 축하 장면처럼 보이는 연출

### 챕터 종료 감정

- `오늘이 지나도, 이건 기억하고 싶다`

## 8. 챕터 5: 조명이 닿지 않는 복도

### 챕터 목적

- 준비 파트에서 감정 파트로 넘어간다
- 과거 기억 조각을 본격적으로 연다
- 팬레터 챕터를 받을 준비를 시킨다
- `방송을 하기 전의 대기실` 같은 공기를 만든다

### 플레이 콘텐츠

1. 백스테이지 문 개방
2. 복도 탐색
3. 기억 조각 3개 회수
4. 정리되지 않은 물건 확인
5. 마지막 문 앞 독백

### 복도 오브젝트

- 오래된 소품 상자
- 낡은 메모
- 작은 액자
- 빛이 깨진 전구
- 프로젝터 또는 모니터
- 이름 없는 선물 포장지

### 기억 조각 구성

기억 조각은 인게임에서 연도를 말하지 않고 아래 감정 층위로 구분한다.

- 처음 크게 축하받았던 기억
- 시간이 쌓이며 익숙해진 축하의 기억
- 아직 온기가 남아 있는 가까운 기억

### 연출 방식

- 조사 시 짧은 화면 전환
- 소리 한 조각, 문장 한 줄, 이미지 한 프레임만 보여 줌
- 다 설명하지 않고 빈틈을 남기는 편이 좋다

### 챕터 종료 감정

- `내가 생각한 것보다 훨씬 오래, 많은 마음이 여기 남아 있었구나`

## 9. 챕터 6: 팬레터를 열다

### 챕터 목적

- 전체 스토리의 핵심 감정 회수
- 예전 생일의 팬레터를 올해의 감정과 연결
- 치즈냥이 축하를 받아들일 준비를 마치게 함
- 방송을 시작하기 직전, 어떤 마음으로 카메라 앞에 설지 정리하게 한다

### 플레이 콘텐츠

1. 우편함 앞 망설임
2. 첫 편지 열람
3. 편지 3~5개 추가 열람
4. 핵심 편지 1~2개 별도 연출
5. 치즈냥 독백
6. 마지막 무대 문 개방

### 편지 구조

팬레터는 전부 긴 문서일 필요 없다.

- 짧은 한 문장 편지
- 3~4줄짜리 짧은 편지
- 핵심 편지 1~2개는 조금 더 길게

### 편지 출처 처리

- 실제 자료를 기반으로 하되
- 닉네임, 개인 정보, 너무 사적인 내용은 정리
- 게임 안에서는 `예전 생일에 받았던 편지`로만 보이게 함

### 편지 묶음 추천

- 고마움 중심 편지
- 응원 중심 편지
- 오래 봐 온 마음이 느껴지는 편지
- 치즈냥이 울컥할 만한 짧은 문장

### 감정 흐름

1. 읽기 전: 망설임
2. 읽는 중: 여운
3. 읽고 난 뒤: 실감
4. 정리 후: 시작할 준비

### 챕터 종료 감정

- `응. 이제 정말 시작할 수 있겠다`

## 10. 챕터 7: 생일 무대

### 챕터 목적

- 현재와 과거를 한 장면으로 묶는다
- 치즈냥이 직접 무대로 나아가게 한다
- 엔딩을 분기한다
- `스트리머 치즈냥`이 오늘의 생일 방송을 여는 장면으로 마무리한다

### 플레이 콘텐츠

1. 마지막 문 개방
2. 완성된 준비방 재확인
3. 무대 진입
4. 최종 독백
5. 엔딩 카드

### 무대 연출 포인트

- 지금까지 켰던 불이 전부 연결됨
- 케이크, 포토존, 책상, 팬레터 감정이 한 화면에서 정리됨
- 기억 조각이 짧게 스쳐 지나가도 좋음
- `이제 송출을 열 수 있다`는 감각이 마지막에 분명해야 한다

### 엔딩 분기 기준

- 준비 완료도
- 기억 조각 회수 수
- 팬레터 열람 깊이

### 엔딩 방향

- 조용한 엔딩: 최소 정리만 한 상태
- 따뜻한 엔딩: 대부분 정리한 상태
- 진심 엔딩: 기억과 팬레터까지 깊게 회수한 상태

### 최종 감정

- `이제는 오늘을 시작할 수 있겠다`

## 11. 챕터 외 보조 콘텐츠

100분을 안정적으로 만들려면 메인 외에 보조 콘텐츠도 필요하다.

### 준비방 자유 조사

- 챕터와 무관하게 읽을 수 있는 짧은 반응 텍스트
- 컵, 의자, 커튼, 장식 상자, 바닥 소품 등

### 기억 조각 수집

- 필수 3개 + 선택 3~5개 정도
