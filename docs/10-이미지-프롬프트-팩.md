# ChatGPT Image Production Guide

- Version: 1.1
- Date: 2026-04-16
- Project id: `zunyang-party-panic`
- Goal of this document: ChatGPT 이미지 생성만으로 게임에 필요한 전체 이미지 세트를 빠짐없이 만들 수 있도록, 세션 시작 문구와 자산별 프롬프트를 한 문서에 정리한다.

## 1. 이 문서의 사용 방식

이 문서는 `ChatGPT 이미지 생성`에 바로 붙여 넣는 작업 지침서다.

사용 순서는 반드시 아래처럼 간다.

1. 새 ChatGPT 대화를 연다.
2. 레퍼런스 이미지 5장을 업로드한다.
3. 이 문서의 `세션 시작 프롬프트`를 먼저 넣는다.
4. 그 다음 `자산 생성 순서`대로 프롬프트를 하나씩 넣는다.
5. 이미지가 마음에 안 들면 `수정 프롬프트`를 추가로 넣는다.
6. 각 결과물을 지정된 파일명으로 저장한다.

중요한 점:

- 모든 이미지는 `같은 대화` 안에서 생성해야 한다.
- 그래야 캐릭터 얼굴, 귀, 머리핀, 색감, 배경 분위기가 계속 유지된다.
- 프롬프트는 한 번에 여러 개 넣지 말고, `이미지 하나당 프롬프트 하나`로 진행한다.

## 1-1. 현재 구현 기준에서 이 문서가 커버하는 범위

이 문서는 `예전 초기 아이디어`가 아니라 `현재 구현 기준`으로 필요한 외부 생성 이미지를 묶는다.

즉, 아래 기준을 따른다.

- 현재 콘텐츠 소스 기준: `src/main/java/org/nowstart/zunyang/partypanic/content/GameContent.java`
- 현재 레퍼런스 기준: `asset/img.png`, `asset/img_1.png`, `asset/img_2.png`, `asset/img_3.png`, `asset/img_4.png`
- 현재 목표: `MVP에서 실제로 쓸 이미지 12종`

중요:

- 예전 문서에는 케이크 베이스, 촛불, 풍선 같은 다른 초기 안이 남아 있을 수 있다.
- 이 문서는 그 초기 안이 아니라 `현재 코드에 들어간 선택지/이벤트 이름`을 기준으로 잡는다.
- 따라서 이 문서대로 만들면 `현재 프로젝트 기준 필요한 이미지 전체 세트`가 나온다.

### 현재 코드 기준 이미지 인벤토리

| 분류 | 개수 | 대상 |
|---|---:|---|
| 메인 캐릭터 | 1 | 생일 방송 호스트 캐릭터 |
| 무대 배경 | 4 | `desk-party-stage`, `mint-cats-stage`, `cake-rush-stage`, `finale-stage` |
| 2차 선택 카드 | 3 | `fan-letter`, `mini-game`, `photo-time` |
| 사고 이벤트 카드 | 3 | `audio-pop`, `cake-balance`, `camera-chaos` |
| 결과 화면 | 1 | 최종 결과 배경 |
| 합계 | 12 | 현재 MVP 외부 생성 필요 이미지 전체 |

### 이 문서에 포함하지 않는 항목

아래는 `별도 일러스트 생성`보다 코드나 UI 도형으로 처리하는 것을 전제로 제외한다.

- 상단 HUD
- 버튼 배경
- 게이지 바
- 결과 등급 글자
- 투표 바와 숫자 UI
- 단순 패널, 프레임, 하이라이트
- 간단한 파티클, 반짝임, 색상 오버레이

## 3. 고정 캐릭터 설정

모든 이미지에서 아래 캐릭터 설정은 흔들리면 안 된다.

- 금발 또는 아주 밝은 크림 블론드 헤어
- 큰 동물 귀
- 빨간 리본이 묶인 옆머리 브레이드 포인트
- 보라색 머리핀과 작은 별 장식 핀
- 붉은 계열 또는 핑크 레드 계열 눈
- 귀엽고 부드러운 한국 VTuber 여자 캐릭터 분위기
- 핑크, 크림, 흰색, 연한 금색 계열 중심

## 4. 전체 아트 방향

이 프로젝트 전체 아트는 아래를 유지해야 한다.

- 귀엽고 따뜻한 한국 VTuber 생일 방송 분위기
- 파스텔 위주이되 흐릿하지 않고 읽기 쉬운 게임용 이미지
- 과한 리얼풍 금지
- 날카로운 메탈릭 SF 느낌 금지
- UI 위에 얹을 수 있게 중심부 가독성 확보
- 방송용이라 화면이 복잡하지 않게 정리
- 고양이, 리본, 풍선, 선물, 케이크, 작은 소품이 잘 어울리는 무드

## 5. 세션 시작 프롬프트

아래 문장을 새 ChatGPT 이미지 생성 대화의 첫 메시지로 넣는다.

```text
I am creating a full cohesive image set for one Korean VTuber birthday fan game. Use the uploaded reference images as the exact style and character identity anchor for everything generated in this chat.

Important character identity:
- female Korean VTuber
- long light blonde hair
- large soft animal ears
- red ribbon tied into a side braid detail
- purple hairpins and small star accessory pin
- warm red or pink-red eyes
- cute, gentle, polished anime VTuber face

Global visual direction:
- cute Korean VTuber birthday livestream mood
- polished anime game illustration
- pastel pink, cream, pale gold, soft coral, and light mint palette
- warm, soft, adorable, celebratory tone
- clean silhouettes for game UI
- no text, no watermark, no logo
- no photorealism
- avoid harsh contrast, avoid sci-fi styling, avoid dark horror mood
- backgrounds must leave enough clean readable space for gameplay UI overlays
- cards must be easy to read at thumbnail size

From now on, keep every image in this same world, same branding, same character identity, and same rendering language.

When I send each next prompt, generate only one image for that prompt and keep style consistency with all previous images in this chat.
```

## 9. 이미지가 잘못 나오면 쓰는 수정 프롬프트

아래는 후속 수정용 짧은 프롬프트다.

### 9-1. 캐릭터 얼굴이 달라졌을 때

```text
Keep the exact same character identity as the uploaded references: blonde hair, large animal ears, red ribbon braid detail, purple hairpins, red or pink-red eyes, and the same polished Korean VTuber face feeling. Do not redesign the character.
```

### 9-2. 화면이 너무 복잡할 때

```text
Make the composition cleaner and reduce small noisy details. Leave more calm open space in the center for game UI overlays. Keep the same style and color palette.
```

### 9-3. 색감이 너무 어둡거나 진할 때

```text
Brighten the image and return to a softer pastel VTuber birthday palette with more cream, soft pink, pale gold, and light mint. Keep the same composition.
```

### 9-4. 카드 이미지가 축소 시 잘 안 읽힐 때

```text
Simplify the image into one strong central focal object cluster so it reads clearly at thumbnail size. Keep the same theme and style.
```

### 9-5. 결과 화면 중앙이 너무 복잡할 때

```text
Preserve the same overall look but open up the center area much more so a large result grade UI can sit there clearly.
```

## 10. 최종 체크리스트

각 이미지를 저장하기 전에 아래를 확인한다.

- 업로드한 레퍼런스와 같은 캐릭터인가
- 금발, 귀, 리본, 머리핀, 눈색 등 핵심 아이덴티티가 유지되는가
- 방송용으로 보기 좋은 밝은 톤인가
- 이미지 중앙이 너무 복잡하지 않은가
- 배경은 UI를 올릴 여백이 있는가
- 카드 이미지는 작게 봐도 한눈에 읽히는가
- 텍스트, 로고, 워터마크가 없는가
- 모든 이미지가 한 게임 세트처럼 보이는가

## 11. 결론

이 문서는 `한 번의 ChatGPT 이미지 생성 세션` 안에서 필요한 이미지 전체를 빠짐없이 만들기 위한 작업 문서다.  
중요한 것은 개별 이미지 화려함보다 `같은 VTuber 세계관`, `같은 캐릭터`, `같은 방송 무드`, `같은 게임 UI 친화성`을 유지하는 것이다.
