# 챕터 구성

- Version: 0.1
- Date: 2026-04-16
- Project id: `zunyang-party-panic`
- Purpose: 각 챕터에서 플레이어가 실제로 무엇을 하게 되는지 정리한다.

## 1. 공통 규칙

- 플레이어는 치즈냥을 직접 조작한다
- 각 챕터는 `탐색 -> 조사 -> 짧은 이벤트 -> 미니게임 또는 정리 파트 -> 마무리 독백` 흐름을 기본으로 한다
- 무엇을 하든 `오늘 생일 방송을 연다`는 축은 계속 살아 있어야 한다
- 실제 인물의 내면을 과하게 해석하지 않고, 장면과 반응 위주로 진행한다

## 2. 챕터 0: 프롤로그 - 불이 덜 켜진 준비방

### 목적

- 오늘이 생일 방송 날이라는 사실을 알린다
- 준비방이 아직 덜 완성됐음을 보여 준다
- 첫 목표를 `방송 책상`으로 유도한다

### 콘텐츠

1. 오프닝 독백
2. 기본 이동
3. 준비방 주요 오브젝트 첫 조사
4. 잠긴 문 확인
5. 방송 책상으로 유도

### 조사 포인트

- 방송 책상
- 케이크 테이블
- 포토존
- 팬레터 우편함
- 장식 창고 문
- 백스테이지 문

### 끝날 때의 느낌

`일단 방송 자리부터 정리해 보자`

## 3. 챕터 1: 방송 책상의 불을 켜다

### 목적

- 치즈냥이 가장 익숙한 방송 리듬을 되찾게 한다
- 오늘 방송의 첫 화면과 첫 공기를 정리한다

### 콘텐츠

1. 책상 주변 조사
2. 소품 정리
3. 오늘의 무드 선택
4. 책상 정리 미니게임
5. 램프 점등

### 핵심 오브젝트

- 마이크
- 컵
- 키보드
- 메모
- 작은 인형
- 오프닝 멘트 메모
- 모니터

### 미니게임 방향

- 기존 방송형 투표 흐름 대신 `책상 정리/무드 세팅`
- 정리 결과가 오늘 방송의 첫 인상처럼 느껴져야 함

### 끝날 때의 느낌

`좋아. 적어도 방송은 시작할 수 있겠다`

## 4. 챕터 2: 장식 창고를 열다

### 목적

- 공간을 확장한다
- 방송 준비에 필요한 소품을 확보한다
- 치즈냥의 취향과 감각을 보여 준다

### 콘텐츠

1. 창고 문을 여는 조건 찾기
2. 창고 진입
3. 장식 조사
4. 필요한 소품 선별
5. 간단한 정리 퍼즐
6. 허브로 복귀

### 핵심 오브젝트

- 리본 상자
- 전구 꾸러미
- 초 상자
- 인형 바구니
- 오래된 생일 배너
- 포장 상자

### 끝날 때의 느낌

`이제 오늘 쓸 건 어느 정도 손에 잡힌다`

## 5. 챕터 3: 케이크가 가운데에 놓이는 순간

### 목적

- 오늘 방송의 중심 장면을 완성한다
- 생일이라는 사실이 화면에 보이게 만든다

### 콘텐츠

1. 케이크 조사
2. 창고에서 가져온 재료 적용
3. 케이크 밸런스 미니게임
4. 완성 컷

### 핵심 오브젝트

- 비어 있는 초 자리
- 기울어진 장식
- 리본
- 접시
- 촛불 조명

### 미니게임 방향

- 기존 케이크 밸런스 게임 유지 가능
- 의미는 `사고 수습`보다 `완성 직전 균형 잡기`

### 끝날 때의 느낌

`이제 화면에 오늘의 중심이 생겼다`

## 6. 챕터 4: 포토존에 오늘을 남기다

### 목적

- 방송 화면에 남길 장면을 정리한다
- 오늘을 기록하고 싶어지는 마음을 만든다

### 콘텐츠

1. 포토존 조사
2. 소품 배치
3. 조명 조정
4. 포토 미니게임
5. 결과 컷 확인

### 핵심 오브젝트

- 카메라 삼각대
- 소품 인형
- 의자
- 배경 천
- 조명
- 바닥 장식

### 미니게임 방향

- 기존 포토 게임 유지 가능
- 의미는 `고득점`보다 `오늘 방송에 남길 컷 정리`

### 끝날 때의 느낌

`오늘 장면 하나는 제대로 남길 수 있겠다`

## 7. 챕터 5: 조명이 닿지 않는 복도

### 목적

- 준비 파트에서 기억 파트로 넘어간다
- 예전 생일 기록 조각을 본격적으로 연다

### 콘텐츠

1. 백스테이지 문 개방
2. 복도 탐색
3. 기억 조각 3개 회수
4. 오래된 물건 조사
5. 팬레터 파트로 연결

### 핵심 오브젝트

- 오래된 소품 상자
- 짧은 메모
- 액자
- 깨진 전구
- 프로젝터
- 선물 포장지

### 기억 조각 방향

- 첫 기억
- 쌓여 온 기억
- 가장 가까운 기억

### 끝날 때의 느낌

`오늘이 갑자기 생긴 건 아니었다는 건 알겠다`

## 8. 챕터 6: 팬레터를 열다

### 목적

- 방송을 켜기 전에 남아 있던 기록을 정리한다
- 예전 생일 편지를 오늘의 시작과 연결한다

### 콘텐츠

1. 우편함 앞 망설임
2. 첫 편지 열람
3. 편지 3~5개 추가 열람
4. 핵심 편지 1~2개 별도 연출
5. 마무리 독백
6. 마지막 무대 문 개방

### 편지 처리 원칙

- 실제 자료 기반이더라도 길게 복붙하지 않는다
- 닉네임, 개인 정보, 사적인 내용은 정리한다
- 인게임에서는 `예전 편지`로만 보이게 한다

### 끝날 때의 느낌

`방송 켜기 전에 보고 갈 건 다 본 것 같다`

## 9. 챕터 7: 생일 방송 무대

### 목적

- 현재와 기억을 한 장면으로 묶는다
- 치즈냥이 오늘 생일 방송을 시작하는 장면으로 마무리한다

### 콘텐츠

1. 마지막 문 개방
2. 완성된 준비방 재확인
3. 무대 진입
4. 최종 독백
5. 엔딩 카드

### 엔딩 분기 기준

- 준비 완료도
- 기억 조각 회수 수
- 팬레터 열람 깊이

### 엔딩 방향

- 조용한 엔딩
- 따뜻한 엔딩
- 진심 엔딩

### 끝날 때의 느낌

`이제 오늘 방송을 시작할 수 있겠다`
